С помощью Git программисты и рпзработчики ориентрируются в коде и отслеживают изменения. Git помогает вернуть файлы в исходное состояние и видеть изменения , внесенные в определенный период. Разработчик выполняет разные команды (например, commit,push), а все изменения синхронизируются с центральным репозиторием. Git - это система контроля версвий , Github - онлайн-сервсис, по сути социальная сеть. Одна из основных целей Github - быть единым местом для проектов с исходным кодом. Предполагается, что польхователь делиться чем то полезным , а другие люди смогут учавствовать в разработке.
